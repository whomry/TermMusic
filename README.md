# TermMusic
Song player with TermMusic

This a Song player based with mpg123, so you must install this package on your computer

#HowToUse

Give permission to file script with:
  chmod +x termumusic.sh

Run as a script with:
  bash termumusic.sh

Or run with a spesific folder with:
  bash termumusic.sh /path/folder/music

Note: change "/path/folder/music" to your music directory
